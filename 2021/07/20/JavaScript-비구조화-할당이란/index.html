JavaScript: 비구조화 할당이란?

Posted on 2021-07-20

안녕하세요, 오늘은 자바스크립트를 훨씬 간단하게 작성하게 도와주는 비구조화 할당에 대해서 이야기를 나누어보려고 합니다.

비구조화 할당(destructuring assignment)이란?

  • 배열이나 객체 속성을 분해해서 그 값을 변수에 담게 해주는 표현

  • 객체 구조 분해라고도 부름

배열

만약 배열을 하나하나 변수로 지정해주고자 한다면 let user1 = users[0], let user2 = users[1]…이렇게 하나하나 작성해주어야 합니다. 만약, 비구조화 할당을 사용한다면 저렇게 일일이 적어줄 필요없이 한 줄로써 작성이 가능해 집니다.

변수를 하나하나 설정해주는 것 보다 훨씬 간단해졌죠?

만약 이런경우는 어떻게 해야할까요? let [a , b, c] = [10,20]

위의 경우는 undefined가c에 들어가게 됩니다. 하지만 undefined를 c에 넣고싶지 않다면, 간단하게 미리 기본값을 설정해줄 수 있습니다. 

```let [a=1,b=2,c=3] = [10,20]```

이렇게 미리 기본값을 설정해준다면 오류가 날 걱정도, undefined값이 들어갈일도 없어집니다. 또한, 기본값이 있다고 하더라도 설정해준 값이 우선시되며, 아무값도 넣지 않았을때만 기본값을 사용하게 됩니다.

여러분은 값을 서로 swap하고싶을때 어떻게 하시나요? 보통은 새로운 변수 c를 선언하여 값을 옮겨담죠. 하지만 구조분해할당을 사용한다면 [a,b] = [b,a] 한 줄로 swap이 가능하게 됩니다. 편리하죠?

객체

배열뿐만 아니라 객체도 비구조화 할당이 가능합니다. 배열과는 달리 객체의 경우 나열된 순서를 신경쓰지 않고 구조분해를 해도 됩니다.

이렇게 비구조화 할당을 하게 된다면 해당 값을 불러울때 user.name이렇게 하지 않고 바로 name으로 값을 불러올 수 있게 됩니다. 함수 내부에서도 사용이 가능해 많이 활용하게 됩니다.

객체는 key : 별명 의 형태로도 사용이 가능합니다. 즉, 새로운 변수 이름을 할당이 가능합니다.

이렇게 새로 할당된 변수 이름과 기존의 이름 모두 활용이 가능합니다. 위의 함수를 풀이해보자면, user의 name에서 값을 가지고와서 userName에 복사해서 넣어줘라는 뜻입니다. age도 마찬가지로 user의 age에서 값을 가지고와서 userAge에 넣음으로써 name과 userName 둘 다 같은 값을 가지게됩니다.

객체도 배열처럼 기본값 설정이 가능할까요?

정답은 ‘네’입니다. 객체도 배열처럼 기본값을 설정하여 사용할 수 있습니다.

또한, 깊숙한 곳에 있는 값을 꺼내올수도 있습니다.

이렇게한다면 훨씬 편리하게 사용이 가능하게 됩니다.

함수의 파라미터또한 객체의 비구조화 할당으로 사용이 가능하고, 모든 파라미터를 꼭 가져오지 않아도 됩니다.

이렇게 비구조화 할당에 대해서 알아보았는데 정리가 되시나요?
