자바스크립트 : Getter & Setter

Posted on 2021-07-19

안녕하세요, 여러분!
오늘은 자바스크립트에서 getter와 setter에 대해서 알아보려고 합니다! 여러분들은 getter와 setter이 필요한 이유와 어떻게 사용하시는지 알고 계신가요? 모르신다면 이번 포스팅을 통해 함께 알아보아요.

프로퍼티

  1. 데이터 프로퍼티
  • 지금까지 사용한 (우리가 흔히 아는) 모든 프로퍼티
  1. 접근자 프로퍼티

  • 본질은 함수

  • 값을 얻고(get) 설정(set)하는 역할을 담당

GETTER와 SETTER이 필요한 이유

일반 데이터 프로퍼티를 사용하다보면 코드가 조금 지저분해보이고 길다고 생각해보신분 계신가요?

이 때 getter와 setter을 사용하면 됩니다. 접근자 프로퍼티를 사용하면 본질은 함수이지만 프로퍼티로 보이기 때문에 (사용하기 때문에) 코드길이가 조금 줄어들고, 외부에서 값을 받아 설정을 할 수 있답니다.

특징

  • get 함수와 set 함수의 이름은 동일해도 괜찮으며 실제 값은 별도의 프로퍼티 (_함수 이름)에 저장됩니다.

    • 예) get 함수 이름 : name() / 실제 값 : _name

  • get은 파라미터가 없는 함수로, 프로퍼티를 읽을 때 작동 (값을 얻어옴)

  • set은 파라미터가 하나인 함수로, 프로퍼티에 값을 쓸 때 호출됨 (값을 설정함)

이렇게 접근자 프로퍼티인 GETTER와 SETTER에 대해 알아보았는데 어떠신가요?
오늘도 즐거운 하루 되세요!
